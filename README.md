## 🛠️ Installation & Setup

### 1. Requirements
Ensure you have **Python 3.10+** installed.

### 2. Download Dependencies
Run the following command in your terminal to install the revolutionary library stack:

```bash
pip install -r requirements.txt
```

#### Detailed Library List:
- **Dash / Dash Bootstrap Components**: High-performance UI framework.
- **OpenCV**: Optical processing and telemetry.
- **MediaPipe**: Real-time 3D pose estimation.
- **Plotly**: Advanced 3D/2D data visualization.
- **NumPy / Pandas**: Data management and array manipulation.
- **SciPy**: Signal filtering and peak detection.
- **Torch / TorchHub**: Deep learning models for monocular depth extension.

### 3. Execution
Launch the master system:

```bash
python app.py
```

Open your browser and navigate to: `http://localhost:8050`
